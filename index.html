Dúvidas (FAQ)

Como alternar modos?

Use os ícones abaixo do relógio: esquerda para cronômetro, direita para timer.

Como definir o tempo?

No modo Timer (ampulheta), clique nos números para digitar minutos e segundos.

Existe atalho de teclado?

Use a tecla 'Tab' para navegar e 'Enter' ou 'Espaço' para ativar os botões.

Como mudar o tema?

Clique no ícone de lua/sol no canto inferior direito para modo escuro.

O que é Alto Contraste?

No menu de acessibilidade (canto superior direito), ative para cores preto e branco.
